A high-fidelity music discovery and hardware management application inspired by the **Soundcore** ecosystem. Designed for audiophiles who crave immersive soundscapes and precision EQ control.

---

## ✨ Key Features

### 🎧 **Deep Discovery**
A modern, dark-themed dashboard featuring curated "New Releases," "Recently Played," and "Trending Now" sections. Experience music with high-impact visuals and seamless transitions.

### 🌊 **Ambient Soundscapes**
A built-in **Ambient Mixer** that lets you create your perfect environment. Toggle and layer sounds like *Soft Rain*, *Ocean Waves*, and *Forest Birds* with individual volume sliders to find your focus or relaxation sweet spot.

### 🎛️ **Device & EQ Management**
A dedicated hardware management interface for the **Liberty 4 Pro**. 
- **Battery Status:** Real-time monitoring of connected devices.
- **Sound Profiles:** Quick-switch between *BassUp™*, *Acoustic*, and *Soundcore Signature*.
- **Visualizer:** A sleek, animated frequency display representing your active EQ.

### 📱 **Universal Player**
A persistent, glassmorphism-inspired player bar with a neon progress tracker, volume management, and detailed track info.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/JJ2025-1/PLAYBOI.git
   cd PLAYBOI
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) to experience the sound.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Utilities:** `clsx`, `tailwind-merge`

- **Background:** `#0a0a0c` (Deep Obsidian)
